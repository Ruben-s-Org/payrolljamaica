## 5. Cross-Linking Rules

These rules govern how the SEO Specialist adds internal links. Follow them on every post update.

### Rule 1: Every Post Links to At Least One Hub Page
No exceptions. Every post that goes live or gets updated must include a contextual link to `/calculator` or `/pricing` (ideally both).

### Rule 2: Minimum 2 Cluster Links Per Post
Each post should link to at least 2 other posts within its topic cluster. Use the cluster groupings in Section 2 above.

### Rule 3: Pillar Pages Get Links From All Cluster Posts
Every post in Cluster A must link to the PAYE pillar. Every post in Cluster B must link to the NIS pillar. Etc. Pillar pages do NOT need to link back to every sub-post — they should selectively link to the most important 5-8 posts in the cluster.

### Rule 4: No More Than 5 Internal Links Per Post
Excessive internal links dilute link equity. Prioritize quality over quantity. For short posts (under 800 words), use 2-3 links. For long posts (1500+ words), up to 5 links is acceptable.

### Rule 5: Place Hub Page Links In-Body, Not Just in CTAs
The CTA button at the bottom of every post already links to `/pricing`. The in-body contextual link to `/calculator` or `/pricing` needs to be added manually in the article text — this is what passes SEO link equity.

### Rule 6: Anchor Text Variety
Do not use the same anchor text for the same destination URL within a single post. Do not use the same anchor text repeatedly across multiple posts that all go live in the same week — vary them using the guide in Section 4.

### Rule 7: March 31 Urgency Posts Link to `/pricing` Exclusively
Countdown posts (1-day, 2-days through 19-days to March 31) should push readers to `/pricing` with urgency-driven anchor text. The calculator is not the right CTA for deadline-urgency content.

### Rule 8: Calculator Posts Link to `/calculator` Exclusively
