### Priority 4: Featured Snippet Optimization (Due: March 24)

Featured snippets (the answer box at the top of Google results) can deliver 20–30% of all clicks for a keyword, even before position #1. We have strong candidates for three high-value queries.

**Target queries:**
- "PAYE threshold Jamaica 2026"
- "how to calculate NIS Jamaica"
- "NHT rate Jamaica 2026"

**Task for each target query:**
1. Find the existing blog post that covers this topic (search the CMS).
2. Add a **Direct Answer Paragraph** within the first 200 words of the article. This paragraph should directly answer the query in 40–60 words, plain and factual. Example for "NHT rate Jamaica 2026": *"The NHT (National Housing Trust) contribution rate in Jamaica for 2026 is 2% for employees and 3% for employers, calculated on gross salary. There is no income ceiling. Contributions are mandatory for all employed workers in Jamaica."*
3. For process-based queries (e.g., "how to calculate NIS Jamaica"), convert the explanation into a **numbered step-by-step list** immediately following the direct answer paragraph.
4. Add **FAQ schema markup** to the post. If using Yoast SEO Premium or Rank Math, use the built-in FAQ block. Add 3–5 Q&A pairs covering the most-asked sub-questions on the topic.
5. Update the meta description to match the direct answer paragraph language.

**Deliverable:** List of 3 posts updated with before/after word counts and confirmation of FAQ schema added, sent to CMO by March 24.
