## Step-by-Step Setup Instructions (30 Minutes)

**Before you start, have ready:**
- Gmail: hello@payrolljamaica.com
- Credit card for billing (can set to JMD billing)
- Google Analytics 4 property ID for PayrollJamaica
- Access to your website to install the conversion tag

---

### Step 1: Create Google Ads Account (3 min)

1. Go to [ads.google.com](https://ads.google.com)
2. Click "Start now"
3. Sign in with **hello@payrolljamaica.com**
4. When asked "What's your main advertising goal?" — click **"Switch to Expert Mode"** at the bottom (critical — avoid the simplified setup)
5. Select goal: **"Website traffic"**
6. Enter website: `payrolljamaica.com`
7. Skip the campaign creation for now — go to account settings first

---

### Step 2: Account Settings (2 min)

1. Click the gear icon → **Settings → Account settings**
2. Set timezone: **UTC-5 (Jamaica Standard Time)**
3. Set currency: **USD** (Jamaica campaigns still bill in USD on Google; JMD billing is unreliable)
4. Save

---

### Step 3: Set Up Conversion Tracking (5 min)

1. Click **Tools & Settings → Measurement → Conversions**
2. Click the **+** button → **Website**
3. Name it: "Calculator Sign-Up"
4. Category: Sign-up
5. Value: J$7,500 (use 48.0 in USD equivalent, or set manually)
6. Count: One
7. Click "Save and continue"
8. Choose: "Use Google Tag Manager" or "Install tag yourself"
9. Add the conversion tag to your site's sign-up confirmation page (`/thank-you` or equivalent)
10. Add a second conversion: "Calculator Use" — Category: Other — Count: Every

---

### Step 4: Create the Campaign (5 min)

1. Click **Campaigns → + New Campaign**
2. Goal: **Website traffic**
3. Campaign type: **Search**
4. Check: "Website visits" → Enter `payrolljamaica.com/calculator`
5. Click "Continue"
6. Campaign name: `Jamaica Payroll March 31 Deadline 2026`
7. Bidding strategy: **Maximize clicks**
8. Set maximum CPC bid limit: **$0.80 USD** (prevents runaway bids)
9. Budget: **$32.00 USD/day**
10. Start date: Today | End date: **March 30, 2026**

---

### Step 5: Campaign Settings (3 min)

1. Networks: **Uncheck "Display Network"** (Search only — critical)
2. Uncheck "Search Partners" for now (enable after 7 days if budget remains)
3. Locations:
   - Click "Enter another location"
   - Add: **Jamaica**
   - Add: **United States** → then click "Advanced search" → add audience segment "Jamaican diaspora" (use Custom Audience in Step 7)
   - Add: **Canada**
   - Add: **United Kingdom**
4. Language: **English**
5. Audience segments: Set to **"Observation"** for now

---

### Step 6: Create Ad Groups (10 min — 2-3 min each)

Repeat for each ad group:

1. Click **+ New Ad Group**
2. Name it (e.g., "Payroll Calculator")
3. Set default max CPC: **$0.60 USD**
4. Enter keywords (copy from the keyword table above, one per line)
5. For each keyword, prefix with:
   - Quotation marks for phrase match: `"keyword"`
   - Brackets for exact match: `[keyword]`
   - Plus signs for BMM: `+keyword +modifier`
6. Click "Save and continue"
7. Create 2–3 responsive search ads per group (copy headlines/descriptions from ad copy section above)
8. In each RSA, pin Headline 1 to Position 1 (click the pin icon)

---

### Step 7: Add Extensions (3 min)

1. Go to **Ads & Extensions → Extensions**
2. Click **+** → **Sitelink Extension** → add all 4 sitelinks
3. Click **+** → **Callout Extension** → add all callouts
4. Click **+** → **Call Extension** → add +1 (876) 123-4567
5. Click **+** → **Structured Snippet** → add Features list

---

### Step 8: Add Negative Keywords (2 min)

1. Go to **Keywords → Negative Keywords**
2. Click **+** → Campaign level
3. Paste the full negative keyword list from this brief
4. Save

---

### Step 9: Review + Launch (2 min)

1. Click **Review campaign** — fix any warnings
2. Verify budget shows $32.00 USD/day
3. Verify end date is March 30
4. Click **Publish**
5. Note: Ads typically enter review within 1 hour and start serving within 24 hours

---

### Step 10: Day 3 Check-In

After 3 days of running:
1. Check **Search Terms Report** (Keywords → Search Terms) — add irrelevant terms as negatives
2. If any keyword has 0 impressions after 3 days, check Quality Score and bid
3. Pause any ad variant with CTR under 1% after 50 impressions
4. Check which ad group has best conversion rate — shift budget there

## Quality Score Optimization

To maximize ad rank and reduce CPC:

1. **Landing page relevance:** Ensure "PAYE", "NHT", "NIS", "March 31", and "Jamaica" appear in the H1 and first paragraph of `/calculator`
2. **Ad relevance:** Each ad group uses keywords that appear in the ad headlines

4. **Mobile experience:** Landing page must pass Google's mobile-friendliness test at [search.google.com/test/mobile-friendly](https://search.google.com/test/mobile-friendly)

## Countdown Ad Customizer (Use This)

Replace Headline 3 in all urgency variants with:

```
{=COUNTDOWN("2026/03/31 23:59:59","en",20)} Left to File
```

This will automatically display:
- "19 Days Left to File" on March 12
- "5 Days Left to File" on March 26
- "1 Day Left to File" on March 30

This feature alone can increase CTR by 15–30% during deadline campaigns.
