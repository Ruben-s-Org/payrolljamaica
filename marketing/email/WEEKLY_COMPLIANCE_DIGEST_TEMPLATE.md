## Usage Guide for the Weekly Digest

### How to fill in the template each week

1. **[DATE]** — The Monday of that week (e.g., "April 7, 2026")
2. **[DEADLINE_NAME]** and **[DEADLINE_DATE]** — Check the compliance calendar. Common deadlines:
   - 14th of each month: S01 monthly return
   - Quarterly S02 due dates: July 14, October 14, January 14, April 14
   - March 31: Annual returns
   - If no deadline that week, use "No filing deadline this week" and reframe as a preparation reminder for the next one
3. **[TAX_TIP_TITLE]** and **[TAX_TIP_BODY]** — Rotate from the example tips above, or write a new one based on common questions, recent blog posts, or seasonal relevance
4. **[BLOG_POST_TITLE]** and **[BLOG_SLUG]** — Pick from the suggested rotation or feature any new blog post published that week

### Deadline Calendar for Quick Reference

| Month | Deadline | What's Due |
|---|---|---|
| Every month | 14th | S01 monthly return (previous month's payroll) |
| July | 14th | Q1 S02 quarterly return |
| October | 14th | Q2 S02 quarterly return |
| January | 14th | Q3 S02 quarterly return |
| March | 31st | SO1 Annual Return + NHT Annual Return |
| April | 14th | Q4 S02 quarterly return |

### Tone Guidelines

- **Professional but approachable** — write like a knowledgeable colleague, not a textbook
- **Direct** — lead with the action or deadline, not preamble
- **Jamaican context** — reference TAJ, NIS, NHT by name; use JMD amounts; acknowledge local realities (portal congestion, accountant availability around deadlines)
- **No fear-mongering** — state penalties factually when relevant, but keep the overall tone encouraging and empowering
- **Keep it scannable** — short paragraphs, headers, bullet points. Most readers are scanning on mobile

### Metrics to Track

- **Open rate target:** >35% (digest emails typically lower than urgency emails)
- **Calculator click-through:** >5%
- **Blog click-through:** >3%
- **Unsubscribe rate:** Stay under 0.5% per send
- **Best performing tips:** Track which tax tips drive the most calculator clicks — double down on those topics

### Integration with Welcome Sequence

- Subscribers who complete the 5-email welcome sequence are automatically added to the weekly digest list (tag: `welcome-sequence-complete`)
- The digest should feel like a natural continuation — they've been educated, now they get ongoing value
- If a subscriber converts to a trial during the welcome sequence, move them to a separate "customer digest" with more product-specific tips
